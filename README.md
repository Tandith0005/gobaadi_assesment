# 🐄 Gobaadi – AI-Powered Livestock Care Platform

A modern, responsive landing page built with **Next.js**, **React**, and **Tailwind CSS** for **Gobaadi**, a platform that connects farmers with licensed veterinarians while leveraging AI to improve livestock healthcare.

## 🌐 Overview

Gobaadi aims to make veterinary services more accessible by allowing farmers to:

- Book licensed veterinarians
- Receive AI-assisted livestock health insights
- Generate personalized livestock management content
- Access educational resources created by veterinarians
- Improve animal healthcare through digital technology

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/Tandith0005/gobaadi_assesment
```

### Navigate into the project

```bash
cd gobaadi_assesment
```

### Install dependencies

```bash
npm install
```

or

```bash
yarn
```

### Start the development server

```bash
npm run dev
```

Open

```
http://localhost:3000
```

---

## 📦 Build for Production

```bash
npm run build
```

Run production server

```bash
npm start
```
